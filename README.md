# Web

Web application design coursework — tour operator platform. Includes use-case diagrams, BPMN process models, ER diagram, UI wireframes, and a full design report.

## Contents

```
Use-case.drawio       — use-case diagram (draw.io source)
bpmn.drawio           — BPMN process diagram
bpmn_simple.drawio    — simplified BPMN diagram
er-diagram.drawio     — entity-relationship diagram
отчет/
  *.tex               — LaTeX report source
  *.pdf               — compiled report
  Use-case.drawio     — use-case diagram copy used in report
  img.zip             — all diagrams and UI screenshots (25 PNG files)
```

## Diagrams

- **Use-case** — actors and system use cases for the tour operator platform
- **BPMN** — business process models (booking flow, bonus burning)
- **ER** — database entity-relationship schema
- **UI forms** — wireframes for 9 main application screens

## Open diagrams

All `.drawio` files can be opened with [draw.io](https://app.diagrams.net/) (desktop or web).
